#  Server Performance Bash Script

A lightweight bash cli tool that shows server performance status in real-time, some metrics are:

 - CPU usage
 - Memory usage
 - Disk usage
 - Top running processes

---

## Some Features

 - No library dependency
 - Useful for quick diagnosis
 - Fast and simple

---

## Technologies

 - Bash

---

## Usage

 ---bash
 ./server-stats.sh
